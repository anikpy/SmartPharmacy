# Quick Reference - Custom Medicines Feature

## 🚀 Quick Start

### For Shop Owners

#### Add a Custom Medicine (5 minutes)
```
1. Login as Shop Owner
2. Go to: /inventory/add/
3. Click "Add as custom medicine" link in info box
4. Or go directly to: /inventory/add-custom/
5. Fill in medicine details
6. Click "Add Custom Medicine"
7. Done! It's now in your shop only
```

#### Use Custom Medicine in Inventory
```
1. Go to: /inventory/add/
2. Search for your custom medicine
3. It shows as "🏪 Your Medicine"
4. Select and add to inventory
5. Set price and other details
6. Submit
```

#### View Your Custom Medicines
```
1. Go to: /inventory/my-customs/
2. See all medicines you've added
3. Shows name, strength, dosage form, price
4. Click "Add Medicine" to add more
```

1. **Local/Regional Brand**

2. **Generic/Unbranded**

## 📋 Field Descriptions

### Brand Name
- The commercial/brand name
- Example: "Aspirin", "Local Para"
- What customers know it as

### Generic Name
- Active pharmaceutical ingredient
- Example: "Acetylsalicylic Acid", "Paracetamol"
- Medical/chemical name

### Strength
- Dose per unit
- Examples: "500 mg", "100 mg/5 ml", "50 mcg"
- Must include unit

### Dosage Form
- Physical form of medicine
- Options: Tablet, Capsule, Syrup, Injection, etc.
- Default: Tablet

### Manufacturer
- Company/source
- Optional field
- Example: "Local Brand Ltd."

### Suggested Price
- Your reference price
- In Taka (৳)
- Used when adding to inventory
- Can be modified per batch

## ⚠️ Important Notes

### What CAN'T Be Done:
- ❌ Add duplicate custom medicines (same brand, strength, shop)
- ❌ See other shops' custom medicines
- ❌ Delete custom medicines (make inactive instead)
- ❌ Edit custom medicine after creation

### What CAN Be Done:
- ✅ Add unlimited custom medicines
- ✅ Use in inventory like master medicines
- ✅ Set custom prices per batch
- ✅ Mark as inactive if needed
- ✅ View history of additions

## 🆘 Troubleshooting

### Problem: Can't find "Add Custom Medicine" link
**Solution**: Check if you're logged in as Shop Owner (not staff)

### Problem: "Medicine already exists" error
**Solution**: You already added this brand+strength. Check "My Customs"

### Problem: Custom medicine not showing in search
**Solution**: 
- Try exact brand name
- Make sure it's set as active
- Check it was created for your shop

### Problem: Other shop can see my custom medicine
**Solution**: This shouldn't happen! Contact admin if it does

## 🎯 Tips & Tricks

### Best Practices:
1. **Use clear names** - Others might eventually see if promoted
2. **Set accurate strength** - Critical for inventory
3. **Include unit** - "500 mg" not just "500"
4. **Set reasonable price** - Used as default
5. **Complete optional fields** - Helps if promoted to master

### Quick Addition:
1. Have info ready before adding
2. Know dosage form (default is Tablet)
3. Estimate price
4. Click "Add Custom Medicine"
5. Done in < 1 minute
